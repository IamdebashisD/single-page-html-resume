# Personal Portfolio Website

This is a simple **HTML-based personal portfolio website** created using semantic HTML5 elements.  
It represents a developer profile with sections like About, Skills, Experience, Projects, Education, and Contact details.

The project is beginner-friendly and focuses on **clean structure and readability**, without using any CSS or JavaScript.
